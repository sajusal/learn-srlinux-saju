## Interface Configuration

```srl

    interface ethernet-1/1 {
        admin-state enable
        vlan-tagging true
        subinterface 10 {
            admin-state enable
            ipv4 {
                admin-state enable
                address 192.168.10.1/30 {
                }
            }
            vlan {
                encap {
                    single-tagged {
                        vlan-id 10
                    }
                }
            }
        }
        subinterface 20 {
            admin-state enable
            ipv4 {
                admin-state enable
                address 192.168.20.1/30 {
                }
            }

                }
```
